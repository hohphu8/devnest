# DevNest 0.2.6

## Highlights

- Fixed Nginx startup failure on projects with longer local domains.

## Improved

- DevNest now generates safer Nginx `server_names_hash_bucket_size` and `server_names_hash_max_size` settings.
- Nginx should no longer fail with `could not build server_names_hash` when starting managed projects.

## Verified

- Passed Rust format check and focused Nginx bootstrap config test.

## Notes

- No database migration or UI workflow changes in this release.
