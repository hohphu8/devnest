# DevNest 0.2.4

## Highlights

- Added service controls directly in the system tray.
- You can now start, stop, restart, and open logs for tracked services without opening the Services page first.

## Added

- New tray `Services` submenu for Apache, FrankenPHP, Nginx, MySQL, Mailpit, and Redis.
- Per-service tray actions: Start, Stop, Restart, and Logs.
- Tray service labels now show current status, such as `Apache - running` or `MySQL - stopped`.
- Logs from the tray now focus DevNest and open the Logs page for the selected service.

## Improved

- Tray service actions refresh the menu after changes so statuses update quickly.
- Invalid tray actions are disabled based on service state.
- FrankenPHP stop from the tray now follows the same cleanup path as the main app service controls.

## Notes

- Existing tray actions still work: Start All, Stop All, Switch PHP Version, Open DevNest, and Exit.
- This release does not add native log previews inside the tray.
