# DevNest 0.2.16

## Highlights

- Added download cache cleanup in Settings.
- Fixed Apache/Nginx PHP-CGI isolation for projects using the same PHP version.
- DevNest now stops managed services when you fully exit the app.

## Added

- Settings now shows download cache usage for runtime, optional tool, and PHP extension downloads.
- You can clear remaining download cache from Settings after install tasks finish.

## Improved

- Successful runtime, optional tool, and PHP extension installs now clean up their downloaded archives.
- Apache and Nginx projects now get dedicated PHP-CGI backends, preventing same-version projects from blocking each other during local HTTP calls.
- Fully exiting DevNest from the tray now stops managed Apache, Nginx, FrankenPHP, Mailpit, Redis, and MySQL services.

## Notes

- Window close still keeps DevNest running in the tray. Use tray exit when you want DevNest and its managed services to shut down.
