# DevNest 0.2.17

## Highlights

- Fixed HTTPS requests from DevNest-managed PHP runtimes when using cURL or OpenSSL.

## Improved

- DevNest now ships a CA certificate bundle and automatically wires it into generated PHP configs.
- Managed PHP CLI, FastCGI, per-project PHP, and FrankenPHP configs now set `curl.cainfo` and `openssl.cafile`.

## Notes

- Restart DevNest-managed PHP/web services after updating so configs regenerate with the new CA bundle.
